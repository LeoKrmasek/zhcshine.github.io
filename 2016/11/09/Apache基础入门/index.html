Apache基础入门

查看apache加载的模块

apachectl -t -D DUMP_MODULES

禁用和启用模块

sudo a2dismod python
sudo a2enmod python

查看版本号

apachectl -v

赋予权限

2.2.x以下版本

Order deny,allow
Allow from all

2.4.x以上版本

Require all granted
